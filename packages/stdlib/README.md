# bricks-ai-stdlib

Standard library of reusable bricks for the [bricks-ai](https://github.com/hemipaska-maker/bricks) execution engine.

Includes 100+ pre-tested bricks for data transformation, string processing, math, validation, encoding, date/time, and list operations.

## Install

```bash
pip install bricks-ai-stdlib
```

Or install together with the core engine:

```bash
pip install "bricks-ai[stdlib]"
```
