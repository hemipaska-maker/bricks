# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Bricks, please report it responsibly.

**Do NOT open a public GitHub issue for security vulnerabilities.**

Instead, email: hemipaska@gmail.com

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will respond within 48 hours and work with you on a fix before public disclosure.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.4.x   | Yes       |
| < 0.4   | No        |
